About Me

Hello! I’m Luka Metias, a student at UMass Lowell exploring web design and user interfaces. This site is my first GUI homework project. My goals are to practice clean, semantic HTML5, and how to style CSS file, and write documentation that clearly explains what I did and why. I’m especially interested in accessibility, typography, and simple layouts that look good on phones and laptops. Outside of class, I enjoy small coding projects, reading about technology trends as well as new cars.

On this page you’ll find an example of structured content with headings, working links, a styled table, an image of a car that’s also a link, and some HTML entities like ©, ♥, →, and  non-breaking spaces. I wrote this content to be meaningful rather than a random list of tags. If a friend provided design feedback or artwork, I would credit them here in the documentation comment at the top of this file. Enjoy browsing!
